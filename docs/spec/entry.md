# エントリーについて

エントリーは、「一次エントリー期間」と「二次エントリー期間」で行われる。
まず一次エントリーを行い、次に二次エントリーを行う。

## 一次エントリー

エントリー時点では参加は確定しない。一次エントリー締切後にアプリ外で抽選を行い、それぞれのエントリーに対して参加優先順位を付与する。
参加優先順位の順に、定員の範囲内で参加権が与えられる。

一次エントリーでのエントリー数が定員を超過した場合、定員をあぶれた分はキャンセル待ちとなる。
以後、キャンセルがあった場合は、優先順位順にキャンセル待ちから繰り上がる。

一次エントリー締切前にキャンセルすることもできる。その場合、そのエントリーは抽選対象とならない。

## 二次エントリー

二次エントリーでは、エントリーに対して、先着順で即時に参加優先順位を付与する。
すなわち、エントリー時点で定員に空きがあれば即時に参加確定となり、空きがなければキャンセル待ちになる。

## エントリー期間の切り替え

エントリーの受付状態には次の状態があり、上から下へ順番に遷移する。
期間の切り替えは管理者が手動で行う。

* エントリー開始前
* 一次エントリー受付中
* 一次エントリー締切、二次エントリー開始前
* 二次エントリー受付中
* エントリー締切

## 定員

イベントの定員は、イベント全体としての定員が設定される。
エントリー期間ごとの個別の定員は設定されない。
